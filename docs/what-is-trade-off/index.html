Trade Off 트레이드 오프

#작성 중 #개념어 #Trade Off
By 탐정토끼(Taehee Kim)

사전적 정의 : 교환, 마치 양팔 저울처럼 한 쪽을 늘리면, 다른 한 쪽을 잃게 된다는 뜻. 여러 언어나 기술에는 각자 장, 단점이 있으니 내 상황에 맞게 올바른 기술을 선택해야한다는 뜻으로 쓰입니다.

이 블로그는 Svelte와 Elder.js로 만들었습니다.
디자인은 PaperCSS를 기본으로 커스텀했습니다.
폰트는 Pretendard를 사용합니다.
Github Pages로 호스팅하고 있습니다.
오픈소스 개발자분들과 무료 호스팅 서비스에 감사드립니다. :)
